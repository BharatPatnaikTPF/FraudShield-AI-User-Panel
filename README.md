# 💳 FraudShield AI – User Panel

An AI-powered banking user interface that detects fraudulent transactions, provides real-time risk analysis, and features an intelligent chatbot for explainable fraud insights.

---

## 🚀 Overview

FraudShield AI is a smart fraud detection system designed for the banking sector.  
This repository contains the **User Panel (Frontend)** that allows users to:

- Monitor transactions in real-time
- Get instant fraud alerts
- Understand risk through AI explanations
- Interact with an intelligent chatbot assistant

---

## ✨ Features

- 🚨 **Real-Time Fraud Alerts**  
  Detect suspicious transactions instantly

- 📊 **Risk Score Visualization**  
  Each transaction is assigned a fraud probability score

- 🤖 **AI Chatbot Assistant**  
  Ask questions like:
    - "Is my last transaction safe?"
    - "Why was this flagged?"
    - "Am I at risk today?"

- 📱 **User-Friendly Dashboard**  
  Clean and modern UI for seamless experience

- 📜 **Transaction History**  
  Track past transactions and flagged activities

- 🔍 **Explainable AI Insights**  
  Understand *why* a transaction is marked suspicious

---

## 🧠 How It Works

1. User performs a transaction
2. AI model analyzes behavior patterns
3. System assigns a **risk score**
4. If suspicious:
    - User gets an alert 🚨
    - Chatbot provides explanation 🤖
5. User can confirm or report fraud
